**Mission:**  
Explore the user registration flow and product search functionality on KaBuM! using a variety of input data to identify validation issues, usability problems, and inconsistencies in search results.

**Focus Areas:**  
- User registration (form validation, error handling, field behavior)  
- Product search (accuracy, relevance, and consistency of results)  

**Test Data Variations:**  
- Valid and invalid CPF numbers  
- Different ZIP codes (valid, invalid, incomplete)  
- Various search terms (correct, misspelled, generic, specific)  

**Activities:**  
- Attempt user registration with different data combinations  
- Observe system responses to invalid inputs  
- Perform searches using varied keywords  
- Evaluate clarity of error messages and feedback  
- Analyze consistency and relevance of search results  

**(What to Check):**  
- Field validation correctness  
- Clarity and usefulness of error messages  
- System behavior with edge cases  
- Search result relevance and consistency  
- Overall usability and user experience
