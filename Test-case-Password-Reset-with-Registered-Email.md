## CT-001 — Password Reset with Registered Email

**Preconditions:**  
User must have a registered account.

**Test Data:**  
- Email: renanlippi@hotmail.com  

**Steps:**  
1. Access the site "https://open.spotify.com/"
2. Access the "Forgot Password" page  
3. Enter a registered email address  
4. Click on "Send Reset Link"  

**Expected Result:**  
- A success message is displayed  
- User receives a password reset email  
- Email contains a valid reset link
