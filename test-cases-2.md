# 🧪 QA Portfolio — Parabank

This repository contains manual test cases designed to validate key functionalities of the Parabank demo application.

## 📌 Application Under Test
- Name: Parabank  
- URL: https://parabank.parasoft.com  

## 🎯 Objective
Validate critical user flows such as login and registration by applying structured test design techniques aligned with industry best practices.

## 🧠 Test Design Techniques Applied
- Equivalence Partitioning  
- Boundary Value Analysis  
- Negative Testing  

## 📂 Test Coverage

### 🔐 Login
- Valid login  
- Invalid password  
- Empty fields  

### 📝 Registration
- Valid registration  
- Invalid email format  
- Missing required fields
